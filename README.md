# 平台跳跃者

一个包含主菜单、关卡、通关与失败流程的 2D 平台跳跃小游戏，基于 Rust

## 模式与操作
- 主菜单：按 N 开始；按 Q 退出。
- 游戏中：
  - 移动：A/D 或 ←/→
  - 跳跃：空格 或 ↑
  - 掉出场景会失败。
- 通关：到达黄色终点旗帜。若存在下一关，会自动进入下一关；否则进入“通关”界面。
- 通关界面：按 N 重新开始；Q 退出。
- 失败界面：按 R 重试该关；M 回主菜单。
